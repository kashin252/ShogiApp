# 将棋アプリ (Shogi App)

React Native (Expo) で開発された、シンプルかつ高機能な将棋アプリケーションです。

## 特徴

*   **対局モード**:
    *   **対人戦 (PvP)**: 1つの端末で交互に操作して対局できます。
    *   **対AI戦**: 内蔵された将棋AIと対局できます。
*   **UI機能**:
    *   **盤面反転**: ボタン一つで盤面を180度回転させ、後手番の視点に切り替えられます。
    *   **レスポンシブデザイン**: 画面サイズに合わせて盤面サイズが自動調整され、スクロールなしで快適に操作できます。
    *   **持ち駒表示**: 相手側の持ち駒は自動的に逆さま（相手視点）に表示されます。
*   **AIエンジン**:
    *   アルファ・ベータ探索、反復深化、静止探索などを実装したカスタムエンジンを搭載。
    *   駒の位置評価 (PST) による局面評価。

## 実行方法

### 必要要件
*   Node.js
*   npm

### インストール

```bash
npm install
```

### アプリの起動

```bash
npm start
```

Expo Go アプリ（iOS/Android）でQRコードを読み取るか、Webブラウザで実行してください。

## 技術スタック

*   **フレームワーク**: React Native (Expo)
*   **言語**: TypeScript
*   **エンジン**: 自作TypeScript将棋エンジン
